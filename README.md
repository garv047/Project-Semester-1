# Project-Semester-1


## What does this Project contains ??
        This is a console-based adventure game built using C.

        The player can explore, fight enemies, and heal.
        
        The game uses:~

                    -Structures to store player data
                    -Pointers to modify values inside functions
                    -Loops for continuous gameplay
                    -Conditions to control program flow

        The program demonstrates core C concepts like:~

                    -Structures
                    -Functions
                    -Pointers
                    -Decision-making
                    -User interaction
                    -Variable updates

        It's simple yet interactive and ideal for beginners in C programming.
